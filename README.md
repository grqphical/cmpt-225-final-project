# Skip List vs B+ Tree Benchmark
This is a benchmark comparing an implementation of a Skip List in C++ to an implementation of an AVL Tree in C++. This repository is a part of our final project for CMPT 225 at Simon Fraser University. 

## Current Benchmarks
- Random Order Insertion
- Fibonacci Sequence Insertion
- Ordered Insertion
- Searching
- Deletion
